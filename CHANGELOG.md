# Changelog

All notable changes to the LearnStatsVis package will be documented in this file.

## [0.1.0] - 2025-12-02

### Added
- Initial release of LearnStatsVis package
- Interactive visualization for P-values and hypothesis testing with biological examples
- Interactive visualization for t-tests comparing two biological groups
- Interactive visualization for ANOVA comparing multiple treatment conditions
- Interactive visualization for confidence intervals with biological parameter estimation
- Interactive visualization for statistical power analysis in biological experiments
- Main application interface to access all visualizations
- Comprehensive documentation and vignettes
- Test suite for package functions
- Example scripts and tutorials
- Makefile for package development tasks

### Features
- P-value visualization using differential gene expression examples
- T-test visualization comparing gene expression between conditions
- ANOVA visualization comparing gene expression across multiple treatments
- Confidence interval visualization for estimating biological parameters
- Power analysis visualization for designing biological experiments
- User-friendly interface with real-time parameter adjustments
- Biological interpretations for all statistical results
- Export capabilities for visualizations (in future versions)

### Documentation
- README.md with installation and usage instructions
- Vignette with detailed tutorial on using the package
- Function documentation for all applications
- Example scripts for quick start
- Project summary documentation
