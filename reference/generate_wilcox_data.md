# Generate sample data for Wilcoxon Ranksum test visualization

Generate sample data for Wilcoxon Ranksum test visualization

## Usage

``` r
generate_wilcox_data(
  n1 = 15,
  n2 = 15,
  location1 = 0,
  location2 = 0.2,
  alpha = 0.05
)
```

## Arguments

- n1:

  Sample size for group 1

- n2:

  Sample size for group 2

- location1:

  Location parameter for group 1

- location2:

  Location parameter for group 2

- alpha:

  Significance level

## Value

A list containing Wilcoxon test results and visualization data
