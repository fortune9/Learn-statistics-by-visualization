# Generate sample data for regression analysis visualization

Generate sample data for regression analysis visualization

## Usage

``` r
generate_regression_data(
  n = 50,
  slope = 0.5,
  intercept = 2,
  error_sd = 1,
  alpha = 0.05
)
```

## Arguments

- n:

  Sample size

- slope:

  Slope of the regression line

- intercept:

  Intercept of the regression line

- error_sd:

  Standard deviation of the error term

- alpha:

  Significance level

## Value

A list containing regression results and visualization data
