# Generate sample data for power analysis visualization

Generate sample data for power analysis visualization

## Usage

``` r
generate_power_data(effect_size = 0.5, n = 30, alpha = 0.05, power = 0.8)
```

## Arguments

- effect_size:

  Effect size (Cohen's d)

- n:

  Sample size per group

- alpha:

  Significance level

- power:

  Statistical power

## Value

A list containing power analysis data
