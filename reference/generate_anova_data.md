# Generate sample data for ANOVA visualization

Generate sample data for ANOVA visualization

## Usage

``` r
generate_anova_data(
  n1 = 10,
  n2 = 10,
  n3 = 10,
  mean1 = 0,
  mean2 = 0.5,
  mean3 = 1,
  sd1 = 1,
  sd2 = 1,
  sd3 = 1,
  alpha = 0.05
)
```

## Arguments

- n1:

  Sample size for group 1

- n2:

  Sample size for group 2

- n3:

  Sample size for group 3

- mean1:

  Mean for group 1

- mean2:

  Mean for group 2

- mean3:

  Mean for group 3

- sd1:

  Standard deviation for group 1

- sd2:

  Standard deviation for group 2

- sd3:

  Standard deviation for group 3

- alpha:

  Significance level

## Value

A list containing the sample data and test statistics
