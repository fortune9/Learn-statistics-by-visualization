# Generate sample data for confidence interval visualization

## Usage

``` r
generate_ci_data(
  true_mean = 0,
  sd = 1,
  n = 30,
  confidence_level = 0.95,
  num_samples = 20
)
```

## Arguments

- true_mean:

  True population mean

- sd:

  Population standard deviation

- n:

  Sample size

- confidence_level:

  Confidence level (e.g., 0.95 for 95

  num_samplesNumber of samples to visualize

A list containing multiple confidence intervals and their properties
